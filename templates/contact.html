Get in touch

We'd love to hear from you! Whether you have suggestions, found a bug, or just want to share how you're using the calculator, feel free to reach out.

Ways to contact us

GitHub Issues

For bug reports, feature requests, or technical discussions, please open an issue on the project's GitHub repository:

NetlifyCalc on GitHub

Feedback categories

  • Bug report: If something isn't working as expected, please describe the issue and your browser/device
  • Feature request: Have an idea for a new feature or improvement? We'd love to hear it
  • Pricing question: If you think our calculations are incorrect, let us know with details
  • General feedback: How can we improve the user experience?

Reporting pricing discrepancies

If you believe the calculator's estimates don't match Netlify's actual pricing:

  1. Check the Attribution page for the pricing data source
  2. Verify against Netlify's official pricing
  3. Open a GitHub issue with your findings and exact calculations

Privacy

This calculator runs entirely in your browser. No usage data is sent to any server. Your calculator states are stored locally and only shared if you explicitly copy and send the URL.
